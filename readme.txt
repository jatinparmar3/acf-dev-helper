ACF Dev Helper adds a live PHP snippet generator to the ACF field group editor.

== Description ==

This plugin adds a right-sidebar helper panel inside the ACF field group editor.

Features:

* Live field list for all detected ACF fields
* Copy-ready PHP snippet preview
* Modern single-style code output
* Smart snippets for image, file, link, gallery, repeater, flexible content, and relationship fields
* Clean, modern admin UI

== Installation ==

1. Upload the `acf-dev-helper` folder to `wp-content/plugins/`.
2. Activate the plugin in WordPress.
3. Open any ACF field group and use the Code Helper box on the right.

== Notes ==

* Repeater and flexible content snippets use the first nested subfield as a placeholder example.
* Image and file snippets adapt to the field return format when it is available in the editor.
